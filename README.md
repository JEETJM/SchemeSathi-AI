# 🌟 Overview

**SchemeSathi AI** is an AI-powered conversational assistant designed to simplify access to government schemes and scholarships.

Thousands of government programs exist across India, but citizens often struggle to identify schemes they are eligible for due to:

* Complex eligibility criteria
* Scattered information across multiple portals
* Lack of personalized guidance

SchemeSathi AI solves this problem by allowing users to simply describe their profile and receive personalized scheme recommendations.

---

# 🎯 Problem Statement

Millions of eligible citizens are unable to benefit from government schemes because they do not know:

* Which schemes match their profile
* Whether they satisfy eligibility requirements
* What documents are required
* Where to apply

A smart AI-based recommendation system can bridge this gap.

---

# 💡 Solution

SchemeSathi AI provides a simple conversational experience.

Users can ask:

> "I am a 21-year-old B.Tech student from West Bengal with family income of ₹2 lakh. Which scholarships can I apply for?"

The AI analyzes:

✅ Age
✅ State
✅ Education
✅ Category
✅ Family Income
✅ User Requirements

and recommends suitable government schemes.

---

# 🤖 Google Gemma Integration

Google Gemma acts as the intelligence layer of SchemeSathi AI.

Gemma is used for:

* Natural language understanding
* User profile extraction
* Eligibility reasoning
* Personalized recommendations
* Human-like AI responses

Gemma helps transform complex government scheme data into simple and understandable guidance.

# 🚀 Installation & Setup

## Clone Repository

```bash
git clone https://github.com/JEETJM/SchemeSathi-AI.git
```

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

## Backend Setup

```bash
cd backend

npm install

npm start
```
